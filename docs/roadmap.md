# Roadmap

## Now

- Static Web/PWA MVP
- Three bundled puzzles
- LocalStorage save and restore
- Quiet sponsor placeholder
- Today's puzzle foundation
- Today's puzzle completion state
- Clear history foundation
- Manifest and service worker

## Next

- Add more puzzle packs
- Improve daily puzzle rotation
- Add streak and weekly progress summaries
- Add richer clear history
- Add accessibility pass for screen readers and keyboard play
- Add update messaging for service worker refreshes
- Test install behavior on iOS Safari and Android Chrome

## Later

- Curated sponsor placements for Japanese companies and local businesses
- Sponsor review policy
- Better difficulty calibration
- Optional theme settings
- Optional cloud sync, only if it can be done without harming the quiet experience

## Not Planned For MVP

- App Store / Google Play build
- AdMob
- Forced video ads
- Deceptive ad UI
- Payments
- Login
- Server integration
- Automatic puzzle generation
